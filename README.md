# 🌦️ Weather App

A simple and responsive Weather App built using HTML, CSS, and JavaScript.

## 🚀 Features

- Search weather by city
- Displays temperature and weather conditions
- Responsive design
- Clean user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Weather API
